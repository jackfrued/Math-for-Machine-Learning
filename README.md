## 机器学习数学基础知识

读书的时候，数学水平一般，喜欢证明但不擅长计算，喜欢探索公式背后的几何意义但不擅长演算中的奇技淫巧。和大多数人一样，看到复杂的数学公式还是会感到畏惧和头疼，内心热爱数学但又常常被数学劝退。随着知识的积累和年岁的增长，开始或多或少的意识到，数学不仅仅是冰冷的数字和公式，它也是打开更广阔世界大门的那把钥匙。在自然科学的诸多领域，数学好几乎意味着上限很高，尤其是人工智能领域的爱好者和探索者，数学不仅帮助我们更好的理解和使用算法，还帮助我们站在更高的视角洞察问题的本质。做这个项目，一方面把学过的数学知识简单梳理一下，另一方面把这些知识分享出来，跟大家一起交流探讨。
